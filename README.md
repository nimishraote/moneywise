# MoneyWise Starter

This is a starter Next.js project zip for the MoneyWise financial literacy app.

What is included now:
- Premium landing page
- Onboarding screens
- Guided assessment UI
- Personalized plan UI
- Lesson split into 2 pages
- About page
- Legal page
- Basic login and signup placeholders
- App shell with menu and footer link back to nimishraote.com

What is not wired yet:
- Supabase auth
- saved user progress
- OpenAI personalization output
- dynamic financial recommendations
- real lesson tools and calculators

Recommended next steps in the next session:
1. Wire Supabase auth
2. Add assessment state persistence
3. Build the personalized plan API route
4. Connect lesson actions to real tools
5. Add OpenAI coach in context
